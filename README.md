# Suno Prompt Engineer

Transform your ideas into technical Suno v5.5 prompts.

## Features
- **Auto Mode**: AI recommends the best settings based on your request.
- **Refine Mode**: Vylepšete své stávající texty o Suno v5.5 tagy.
- **History**: Sledujte své předchozí generace.
- **Themes**: Vyberte si z několika vizuálních motivů.
- **Album Art**: Generování obalů alb pro vaše písně.

## Setup
1. Copy `.env.example` to `.env`
2. Add your `GEMINI_API_KEY`
3. Run `npm install`
4. Run `npm run dev`
